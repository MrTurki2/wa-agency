# غلا — استراتيجية الانتشار والنمو

> اسم البوت: **غلا** 🌸 — سكرتيرتك الذكية على واتساب
> الهوية: أنثوية، شخصية، عربية أصيلة — باسم بنت المؤسس

---

## الهوية الكاملة

```
الاسم:       غلا
الشخصية:    سكرتيرة ذكية، دافئة، عربية
اللهجة:     نجدية — مش رسمية، مش ثقيلة
الصوت:      ثقة + حنان + ذكاء
التوقعية:   "كأنها تعرفك"

Tagline:    غلا — سكرتيرتك الذكية
Sub:        دايم معاك على واتساب
```

**لماذا "غلا" تحديداً:**
- اسم حقيقي لبنت حقيقية = قصة يحكيها الناس
- أنثوي = ثقة أعلى، نبرة أدفأ
- قصير = ينذكر، يُنطق بسهولة
- "والله غلا ساعدتني" — جملة طبيعية ١٠٠٪

## الشخصيات الثلاث — Personas

### Persona A — الاجتماعي
```
من هو:    علاقاته واسعة، يحب يتواصل، ٥٠+ رقم في جواله
يبغى:     يرسل دعوات، ينظم مناسبات، يتواصل بسهولة
قيمته:    لا يدفع — لكنه يجيبلك ناس. شبكته = تسويقك
مساره:    قوالب دعوات، إرسال جماعي، تذكيرات مناسبات
خطر:      يمل بسرعة لو ما وجد جديد
```

### Persona B — العملي
```
من هو:    مدير، صاحب محل، موظف متقدم
يبغى:     توثيق، تقارير، ربط أنظمة، تنفيذ مهام
قيمته:    يدفع لو وجد قيمة واضحة
مساره:    تحليل فواتير، تقارير مبيعات، إرسال إيميلات، ربط ERP
خطر:      صبره قصير — لو ما نفع من أول جلسة يرحل
```

### Persona C — العائلي
```
من هو:    أب/أم، مهتم بالعائلة والمجتمع
يبغى:     دعوات، تهاني، إدارة مجموعات العائلة
قيمته:    وفي، يبقى طويل، يشارك داخل العائلة
مساره:    قوالب مناسبات، أذكار صباحية، تنبيهات مهمة
خطر:      استخدام متقطع — مو يومي
```

## سيناريوهات الانتشار

### سيناريو ١ — الدعوة الجماعية (الأقوى للانتشار)

```
المستخدم: "أبغى أعزم أخواني على عشاء الجمعة"

غلا: عندك أرقامهم أو تبغى تكتبها؟

المستخدم: عندي — عائلة العم

غلا: لقيت ٨ أرقام في جهاتك باسم "عائلة العم"
     اختار القالب:
     ١- دعوة رسمية
     ٢- دعوة ودية
     ٣- اكتب رسالتك

المستخدم: ٢

غلا: هلا وغلا 🌸 نورونا الجمعة عشاء
     عند أخوكم [الاسم] الساعة ٨ مساءً
     يرسل لك العنوان
     تم الإرسال لـ ٨ أشخاص ✅
```

**لماذا ينتشر:** كل من يستلم الرسالة يرى في الأسفل:
```
أُرسلت عبر غلا 🌸 — wa.me/غلا
```

**المعادلة:** مستخدم واحد يرسل لـ ٥٠ = ٥ مستخدمين جدد (١٠٪ فضول)

---

### سيناريو ٢ — الواو الأولى (تحليل الصور)

```
المستخدم: [يرسل صورة فاتورة]

غلا: فاتورة من مطعم البيك 🍗
     التاريخ: الأحد ٢٣ مارس
     المبلغ: ١٨٧ ريال
     تبغى:
     ١- أحفظها في مصاريفك
     ٢- أرسلها للمحاسب
     ٣- تجاهل

المستخدم: ١

غلا: تم الحفظ ✅ مصاريفك هذا الشهر: ١٢٤٠ ريال
```

**لماذا "واو":** ما توقع إنها تفهم بدون ما يكتب أي شيء

---

### سيناريو ٣ — السكرتيرة التنفيذية

```
المستخدم: "ذكرني باجتماع مع أحمد بكرة الساعة ١٠"

غلا: تم ✅ راح أذكرك الساعة ٩:٣٠
     تبغى أرسل لأحمد تأكيد؟

المستخدم: إيه أرسله

غلا: [إلى أحمد]
     هلا أحمد — موعد الاجتماع غداً الساعة ١٠
     يؤكد معك [الاسم] ✅
```

---

### سيناريو ٤ — الشخص الاجتماعي (النمو الفيروسي)

```
المستخدم: أبغى أرسل تهنئة رمضان لكل جهاتي

غلا: عندك ٢٨٦ رقم. تبغى الكل أو مجموعة معينة؟

المستخدم: الكل

غلا: اختار تصميم التهنئة:
     [صورة ١] [صورة ٢] [صورة ٣]

المستخدم: [يختار]

غلا: تم الإرسال لـ ٢٨٦ شخص 🎉
     ملاحظة: بعض الأرقام الدولية قد تتأخر
```

**التأثير:** ٢٨٦ شخص يستلمون رسالة تحتوي اسم "غلا" في التوقيع

## الـ Onboarding — ٣ نسخ للاختبار

### نسخة A — المباشر
```
غلا: هلا! أنا غلا 🌸 سكرتيرتك الذكية
     باختصار: اسمك الأول؟

[اسم]

غلا: أهلاً [اسم]! إيميلك؟ عشان أرسل بالنيابة عنك

[إيميل]

غلا: ممتاز! أنا جاهزة 💪
     قول لي: وش تبغى؟
```

**تقيس:** سرعة الإتمام، معدل الإكمال

---

### نسخة B — الشخصي
```
غلا: هلا والله! أنا غلا 🌸
     قبل ما نبدأ — كيف أناديك؟

[اسم]

غلا: عيني [اسم] 😊
     أنا هنا أساعدك في أي شيء —
     ترسل رسائل، تحلل صور، تذكرك، تراسل بالنيابة عنك
     باختصار: سكرتيرتك الذكية على واتساب
     نبدأ؟ أعطني إيميلك عشان أقدر أرسل بالنيابة

[إيميل]

غلا: خلاص، أنا جاهزة 💪 وش تبغى؟
```

**تقيس:** هل النبرة الدافئة ترفع الـ D1 retention؟

---

### نسخة C — الـ WOW أولاً (الأذكى)
```
غلا: هلا! قبل ما أعرّف عليك نفسي —
     أرسل لي صورة أو اسألني أي شيء 👀

[المستخدم يجرب]

[غلا تتألق وتجاوب]

غلا: شفت؟ 😊 هذا أنا — أنا غلا
     سكرتيرتك الذكية على واتساب
     اسمك الأول؟
```

**الفكرة:** التجربة قبل الالتزام = انخفاض الـ drop-off

**تقيس:** هل التجربة أولاً تزيد الـ D7 retention؟

## جدول قياس الـ Onboarding

| المقياس | الوصف | الهدف |
|---------|-------|-------|
| Completion Rate | كم أكمل التسجيل كاملاً | > ٧٠٪ |
| Time to Complete | متوسط وقت الإتمام | < ٩٠ ثانية |
| D1 Retention | من رجع بعد يوم | > ٥٠٪ |
| D7 Retention | من رجع بعد أسبوع | > ٣٠٪ |
| First Action | أول طلب بعد التسجيل | يكشف الشخصية |
| Share Rate | من شارك غلا لغيره | > ١٠٪ |
| Drop Point | أين يتوقف الناس | يحدد المشكلة |

## كشف الشخصية من الـ Onboarding

غلا تسأل سؤالاً واحداً ذكياً بعد التسجيل:

```
غلا: سؤال سريع — وش أهم شيء تبغى أساعدك فيه؟

١- أرسل رسائل وأتواصل مع الناس
٢- أنظم شغلي ومهامي
٣- أدير عائلتي ومناسباتي

[يختار]
```

**النتيجة:**
- اختار ١ → Persona A (اجتماعي) → مسار الدعوات والإرسال
- اختار ٢ → Persona B (عملي) → مسار المهام والتقارير
- اختار ٣ → Persona C (عائلي) → مسار المناسبات والتهاني

**من هنا:** شخصية غلا تتكيف مع الشخص

## استراتيجية النمو حسب الشخصية

### Persona A — النمو الفيروسي
```
الخطة:    أعطه خدمة مجانية كاملة للإرسال الجماعي
السبب:    كل رسالة يرسلها = إعلان مجاني لغلا
القياس:   كم مستخدم جاء من كل "رسالة جماعية"؟
الهدف:   كل مستخدم اجتماعي يجيب ٣+ مستخدمين
```

### Persona B — النمو المدفوع
```
الخطة:    ١٤ يوم مجاناً كاملاً ← بعدها اشتراك
السبب:    يعتمد عليها ← لا يقدر يتركها
القياس:   كم أجرى عمليات مدفوعة بعد التجربة؟
الهدف:   ٣٠٪ من B يحولون لاشتراك
```

### Persona C — النمو العائلي
```
الخطة:    مجاني دائماً — قيمته في الشبكة
السبب:    يشارك داخل العائلة ← ينمو عضوياً
القياس:   عدد المجموعات العائلية المرتبطة
الهدف:   كل عائلة = ٥-٨ مستخدمين جدد
```

```

٢١-٥٠    Persona A (اجتماعيون)         تشغيل الـ viral loop
```
